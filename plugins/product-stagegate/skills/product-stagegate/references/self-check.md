# 阶段定位自检 + 止损规则

## 存量项目自检表（定位当前在哪一关）

按顺序回答，第一个"不满足"的问题就是你当前所在阶段。

- **Q1 你有没有系统访谈过目标用户、确认痛点真实且强烈？**
  - 没有 / 只问过身边人 → **Demo 阶段**，先补访谈
  - 做过，≥60% 确认 → 进 Q2
- **Q2 你定义了「激活」的具体行为、激活率 ≥30%，且做过支付意愿测试（Fake door/预购/价值承诺）、转化率 ≥10% 吗？**
  - 还没定义激活 / 激活率 <30% → **MVP 阶段**，专注提升激活
  - 激活达标但没做支付意愿测试 → **仍在 MVP**，补做再判断
  - 激活和支付意愿均达标 → 进 Q3
- **Q3 你的 Day30 留存曲线趋平了吗？Sean Ellis 做了且 ≥40% 吗？**
  - 留存还在跌 / Sean Ellis <40% → **PMF 阶段**，先解决留存
  - 两者达标 → 进 Q4
- **Q4 你有没有 ≥1 个渠道 LTV/CAC ≥3、回收期 <12 月？**
  - 没有 / 还没系统测渠道 → **GTM 阶段**，跑渠道单位经济
  - 有渠道跑通 → 进 Q5
- **Q5 你的自然增长（非付费）占比 >30% 吗？**
  - 不到 30% → **MTU 阶段**，设计传播机制
  - 超过 30% → 飞轮已启动，进精细化运营

> 关键：定位靠这套问题，不靠"我感觉做到哪了"。

## 止损规则（什么时候停下来重新想，而不是无限重试）

### 触发"方向复盘"的条件（满足任一）

- **Demo**：完成 20+ 访谈后痛点确认率始终 <40%，且找不到更聚焦的目标用户群
- **MVP**：2 轮以上迭代（各 4 周）激活率始终 <20%；或激活达标但支付意愿经 2 轮测试始终 <5%
- **PMF**：运营满 3 个月，留存曲线从未趋平，Sean Ellis 持续 <25%
- **GTM**：测 5+ 渠道、超过 3 个月，无一渠道 LTV/CAC >1.5

### 方向复盘会的产出（不是"决定放弃"，而是必须给出明确决定）

1. 重新审视核心假设：哪个假设被证伪了？
2. 识别转型方向（Pivot）：换问题、换用户群、还是换解决方案？
3. 评估继续/停止：当前资源下，转型还是砍掉？

> 没有"继续做下去"这个选项——方向复盘会必须输出明确的"转型方向"或"停止"。
