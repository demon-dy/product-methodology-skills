# 价值模型

## 产品价值公式

```
产品价值 = 功能价值 + 情绪价值 + 资产价值
```

价值必须由**交易对手**决定，不是你自己认为有价值（《项链》玛蒂尔德：用巨大代价展示自我认为的价值，买方并不买单）。

### 功能价值

满足基础需求和效率需求，可标准化、可量化、可比较（更高效/便宜/稳定/安全/易用）。功能品竞争常走向效率和成本。四类功能模型：

- 原材料/劳动力模型：靠资源和劳动交付
- 专利/IP 模型：靠知识产权、技术壁垒、独特内容
- 平台/供应链模型：组织供需、提升流通效率（SHEIN 小单快反）
- 基础设施模型：为大量个人/组织提供底层能力（成为基础设施后被社会责任约束）

> 只做"匪兵甲"式纯功能工具人，功能失效即无价值，陷入残酷可替代竞争。

### 情绪价值

```
情绪价值 = 生理唤起 + 认知标记 + 心理账户
```

用户不为物本身付费，为被唤起、被标记、被安放的感受付费。三个付费点：**保障感**（对抗担心）、**愉悦感**（对抗枯燥）、**彰显性**（对抗自卑/表达身份）。情绪价值也需持续交付，单纯刺激会快速耗尽（貂丁透支即失效）。

### 资产价值

可持续变现才有资产价值，需两个支撑：①有专门市场或二手市场及配套服务；②价格依赖可维护的共识。核心诉求是保值、增值、流通、可退出，不是即时消费。没有二手市场和共识，资产价值退化为情绪消费（泡泡玛特二级市场失效则退化）。钻石：不是材料贵，是共识贵。

## 产品创新 = 价值组合重组

创新不是凭空出现，是新功能/新情绪/新资产，或在原有价值上叠加新价值。常见组合：

- 功能 + 情绪 → 消费/体验升级
- IP + 愉悦感 + 资产 → 潮玩、游戏道具、轻资产收藏
- 原材料稀缺 + 彰显性 + 资产 → 奢侈品
- 功能 + 保障感 → 老牌、信任
- 产品价值 + 辨识度 + 行业领导力 → 头牌

**风险**：只有情绪没功能锚点易脆弱；只有功能没品牌/情绪/关系易被价格战吞没；资产价值无二级市场和共识会退化。

## 品牌层级（定价权）

| 层级 | 用户关系 | 核心能力 | 风险 |
|---|---|---|---|
| 白牌 | 无须记名字 | 原材料+供应链效率 | 被渠道压价，成渠道肥料 |
| 老牌 | 熟悉带来信任 | 长期稳定交付 | 易老化，缺新鲜感 |
| 网红 | 新鲜感+话题 | 抓注意力、制造传播 | 热得快冷得也快 |
| 大牌 | 辨识度+情感 | 持续一致的产品和姿态 | 需长期投入和克制 |
| 头牌 | 行业领导力 | 定义行业方向 | 极少数能做到 |

定价权不是话术，是用户心智中的记忆、情绪、信任、共识可被货币化。
