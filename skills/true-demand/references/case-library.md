# 案例库

当用户要求类比、复盘、案例启发时读取。每个案例不是孤立故事，是某个方法论节点如何工作。

## 价值

| 案例 | 方法论作用 | 可复用判断 |
|---|---|---|
| 《项链》玛蒂尔德 | 价值由交易对手决定 | 不要用自我感动替代买方需求 |
| 超市货架 | 用户不行动是常态 | 研究用户为什么拿起又放下 |
| 塑料袋/包包/爱马仕 | 功能/情绪/资产三类价值 | 付费可能是为身份、投资预期 |
| SHEIN 小单快反 | 供应链也是功能价值 | 效率优势改变品类竞争结构 |
| 匪兵甲 | 纯功能可替代 | 只做工具人陷入残酷竞争 |
| 钻石 | 稀缺/彰显/资产共识被打造 | 奢侈品不是材料贵，是共识贵 |
| 泡泡玛特 | 价值组合创新品类 | 二手市场失效，资产价值退化 |
| 尼彩手机 | 伪价值回应真需求 | 短期收割，长期崩塌 |
| 小米/苹果 | 极致单品/头牌定义范式 | 行业领导力来自大宗师产品 |

## 共识

| 案例 | 方法论作用 | 可复用判断 |
|---|---|---|
| 无抗猪肉 | 感知设计缺失 | 价值点必须被稳定感知 |
| 深圳女孩征婚 | 画像不适合弹性需求 | 关系/情绪/资产产品需人设 |
| 校内网放假大巴 | 强场景完成拉新激活 | 找周期性强场景 |
| 亲子游泳馆 | 切细场景造强感知 | 黄金场景比泛大市场重要 |
